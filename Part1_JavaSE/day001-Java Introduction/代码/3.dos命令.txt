一. 如何打开dos窗口
    1. 按下 windos键 alt  ctrl 之间的键  
      或者打开开始 菜单  输入 cmd 名令

	2. win + r 键 输入 cmd 打开 dos窗口
	
    3. 在地址栏 直接输入 cmd命令  dos窗口 直接定位到当前位置

二.常用命令
	1. 切换盘符  盘符：  E: d: 
	

	
	2.dir命令 列出当前文件夹下所有的内容 
	           包含 文件和文件夹

	3.cd 命令 进入一个目录/文件夹
    
    4. tab  q左边的键   自动补齐文件或者文件夹的名字
	
	5. cd .. 返回上一目录
	
	6. cd / 回到根目录
	
	7.cls 清除屏幕
	
	8. del 命令 删除文件  不经过回收站
	9. del *.doc 删除所有以 .doc结尾的文件
	
	10. del *.* 删除所有的文件
	
	11. exit 命令 退出 dos窗口
	
	12. 可以使用 键盘上的 上 下 键 来找到 之前输入的dos命令
